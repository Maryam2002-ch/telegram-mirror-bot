# 🪞 Telegram Mirror Bot

A simple Telegram bot that echoes everything you send to it.

## ✨ Features

- Echo text messages
- Return photos, videos, animations, stickers, voice messages, and documents
- Clean and modular code

## 🚀 Setup

1. Clone the repository
2. Create a `.env` file: token=YOUR_BOT_TOKEN
3. Install dependencies:
```bash
pip install pyTelegramBotAPI python-dotenv
```
4. Run the bot:
```bash
python echo_telegram_bot.py
```
## 🛠️ Technologies
Python 3

pyTelegramBotAPI (telebot)

python-dotenv
